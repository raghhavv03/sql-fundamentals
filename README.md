# SQL Fundamentals

A structured collection of lessons, assignments, and notes covering core SQL concepts using MySQL. This repository documents my learning journey from basic queries to joins, subqueries, views, and transactions.

## Topics Covered

- Database & table creation (DDL)
- Data manipulation — INSERT, UPDATE, DELETE (DML)
- Data retrieval — SELECT, filtering, sorting (DRL)
- Constraints (PRIMARY KEY, FOREIGN KEY, UNIQUE, CHECK, etc.)
- Referential integrity & relationships between tables
- Joins (INNER, LEFT, RIGHT, FULL, SELF)
- Set operations (UNION, UNION ALL, INTERSECT, EXCEPT)
- Subqueries
- Views
- Transactions (COMMIT, ROLLBACK, ACID properties)

## Repository Structure

```
sql/
├── lessons/          # Topic-wise SQL scripts (01_introduction.sql – 10_transactions.sql)
├── assignments/       # Practice questions, solutions, and assignment PDFs
└── notes/             # Lecture notes and a SQL cheat sheet (PDF)
```

## Tools Used

- MySQL Server
- MySQL Workbench
- Git & GitHub

## How to Use

Each `.sql` file in `lessons/` is self-contained and includes inline comments explaining the concept being demonstrated. Run them sequentially in MySQL Workbench (or any MySQL client) to follow the learning progression from basic queries to transactions.

## Purpose

This repository serves as a personal reference and practice log for building a strong foundation in SQL — covering the querying and database design skills needed for data analysis and backend development.
